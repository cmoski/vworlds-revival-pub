User Interactions

The User Interactions lessons offer a preliminary look at how user interfaces function within a virtual world, and how you can create objects that users can interact with.
a We will look at methods that receive input from users by their navigation in three-dimensional space, their use of the mouse, their text input, and their interactions with external web pages.
a We will also look at how to make objects react to user input with scripts.

At this stage, you can use scripts to write simple methods that can receive messages from world users, and alter the properties of an object. By using VBScript or Jscript/javascript, a world scripter can manipulate any object in the virtual world in real time. If you need a more detailed description of objects, please see the Object Model Overview topic in Virtual Worlds Help. If you would like to read further about object methods, please see the Virtual Worlds Methods topic in the Help."
