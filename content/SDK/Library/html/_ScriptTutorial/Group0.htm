Introduction

Welcome to the Virtual Worlds Scripting Tutorial.

Select a lesson from the bar on the left and follow each step. Each lesson references objects and behaviors that occur in a corresponding 3D view in Virtual Worlds. The lessons include references to sample VBScript files and snippets that you can use as templates for creating and enhancing your own objects and behaviors. For convenience, this tutorial displays the Virtual Worlds Command Window for inputing scripts and displaying messages resulting from executing scripts.

When you click on a topic in the lesson, the objects related to it appear in the 3D view. Following is a list of definitions for the icons used throughout the tutorial:

$ Means you should take note.
a Represents a task.
~ Alerts you to something that is important.
Blue text links into the Virtual Worlds Help Documents.

If you need specific technical information about scripting objects in a virtual world, see the Programmer's Guide in Virtual Worlds Help.

Before starting the lessons, please familiarize yourself with the Virtual Worlds Authoring Tools Tab. You can access the authoring tools at any time by clicking the wrench icon at the top right corner of the Virtual Worlds window. In particular, these tutorials require an understanding of the Object Explorer, one of the advanced tools. You can access the Object Explorer by going to the author tools and clicking the Object Explorer link under the "Advanced section." Before proceeding to the next step, open the Object Explorer.
