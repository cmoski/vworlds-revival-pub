StateMachine Action Help

Any of the following can be pasted into the StateMachine editor VBScript field:

this.SetAState "Temperature", "warm" - Set a state property.

this.DoGesture 3 - Do a gesture if the object has a sprited (.spr) geometry.

this.Say "hi there" - Say something.

this.GetMoveToPosition(0.0, 1.0, 2.0, this).Do - Move to (0.0, 1.0, 2.0).

World.ShowHTML "Hello!" - Display HTML in a pop-up window.

World.ShowURL "http: //vwg/foo.htm" - Display a URL in a pop-up window.

Use right-click to copy and paste (control-C & control-V don't work).
