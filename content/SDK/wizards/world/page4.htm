Step 3 of 4
To host your world on the Internet you will need to do the following:
1.
Setup a virtual directory on a Web server. Use Microsoft FrontPage Personal Web Server, or Microsoft Internet Information Server or your own server setup.
2.
Specify the Content Server location for the world. Make sure your world content is available at this location.
Content server:
For example: http://www.microsoft.com
(If you choose not to specify a Content Server at this time, you can set it later using Microsoft Virtual Worlds Object Explorer.)
3.
To provide others with your world name and server location, use the 'Show in Directory Services' command available under the Tools menu in Microsoft Virtual Worlds Server.
