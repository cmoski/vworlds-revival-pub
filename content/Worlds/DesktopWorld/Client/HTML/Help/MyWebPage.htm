My Web Page

In Desktop World, you can create your own personal web page for other members to see. My Web Page is located on the My Things menu.

Your web page is place where you can express yourself by sharing photos and graphics, favorite web sites and other things you'd like members to know about you.

You can also see other members' web sites by clicking the members name and clicking View Web Page.



To create your web page
  1. Click the My Things tab.

    The My Things menu appears with the My Notes submenu open by default.

  2. Click My Web Page.

    The web page creation tool appears in the Web pane.

  3. Type information about yourself in the About Me and Other Information text boxes.

  4. You can elect to let others see your hotmail address on your web page by clicking Display my hotmail address.

    Note: You will need to enter your hotmail address in My Details first.

  5. Click View Page to preview what you have done.

  6. Close the web page to get back to the web page tool.

  7. Click Save Chages so that other members can see your web page.

    Note: you will need to save your changes anytime you change the text on your web page.



To add a picture to your web page
  1. To include a picture, click Upload a new picture.

    This will launch a dialogue box where you can browse pictures available on your PC.
    Note: you can only upload gif and jpeg file formats.

  2. You can add a caption under your picture by typing into the caption box.

  3. Click Save Changes at the top of the page.

  4. Click View Page to preview how it looks.

  5. You can remove or change your picture at anytime by clicking Delete picture and uploading a new picture.

To add favorite web links
  1. Type the name of your favorite web link. Example: Yahoo!

  2. Enter the full web address of the web link. Example: http://www.yahoo.com

  3. You can share up to four favorite web links.

  4. Click Save Changes at the top of the page.

  5. Click View Page to preview how it looks.



To create a picture album
  1. Scroll down to the bottom of the page where you will find My Picture Album

  2. Click Upload picture 1 and click Browse to select the picure you want to add.

  3. Click OK.

  4. Type your caption in the text area.

  5. You can add up to four pictures in your album.

  6. This pictures will appear at the bottom of your web page.
