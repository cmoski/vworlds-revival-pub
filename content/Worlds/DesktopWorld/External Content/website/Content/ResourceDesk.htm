Resource Desk
At the Resource Desk you can find helpful information about the resources available at the Virtual Worlds Group (vworlds.org) web site. Here are some quick web links to help get you started.
  • Read the FAQ is where you can learn more about the Virtual Worlds Group, common source questions and licensing.
  • Virtual Worlds SDK Tour walks you through features in the application.
  • Documentation includes links to online resources and papers.
  • Report a Bug contains a form for reporting bugs to the Virtuals Worlds Group organization. You can also search the known bug database on this page.
