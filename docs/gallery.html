Available Worlds

Gallery — An art gallery space with rooms for displaying objects
Home — A residential space
Landscape — An outdoor environment
Office — A workspace environment
DesktopWorld — The flagship world with lobby (60+ .X models), music garden, walls, ceiling, patio, tables, and furniture. Based on the original SDK wizard kit coordinates.

All worlds are created from the original SDK wizard kits at Local Content\SDK\wizards\world\Kits\ using the headless VBScript creation scripts.
