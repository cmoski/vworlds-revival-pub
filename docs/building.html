Prerequisites

Visual Studio 2022 (Professional, Community, or BuildTools)
MFC/ATL for v143 build tools — install via VS Installer > Individual Components
Windows SDK 10.0 (any recent version)
No external DirectX SDK needed — legacy D3DRM headers/libs are bundled in third_party/d3drm_legacy/x86/

CMake is bundled with VS2022. If MFC is only in an older toolset (e.g., 14.36), CMake auto-discovers it including spectre-mitigated libs.
Build Commands 

# Configure (must be Win32/x86 - code casts pointers to long)
cmake -B build -G "Visual Studio 17 2022" -A Win32 -DVWORLDS_BUILD_GRAPHICS=ON

# Build all targets
cmake --build build --config Debug

# Build individual targets
cmake --build build --target vwsystem --config Debug   # Core platform DLL
cmake --build build --target vwmm --config Debug       # Graphics/multimedia
cmake --build build --target renderhost --config Debug  # 3D render client
cmake --build build --target serverV2 --config Debug    # Server manager
Build Outputs (Debug)

Target Output Description
vwsystem vwsystem.dll (~3.9MB) Core: object model, server, client, networking, persistence, scripting
vwmm vwmm.dll (~2.6MB) Graphics/multimedia: D3DRM rendering, sprites, geometry, sound, actor animation
vwfound vwfound.dll (~580KB) Foundation exemplars (Thing, Avatar, Room, Portal)
vwprops vwprops.dll (~457KB) Property COM server (Vector, Sound, PickData, AccessoryData, IntersectData)
vwstudio vwstudio.dll (~1.2MB) Studio authoring tools (geometry, boundary, web, undo)
vwrendvw vwrendvw.dll (~780KB) Render View OCX (3D viewport control)
objexplr objexplr.dll (~700KB) Object Explorer (MFC OCX control)
uictrl uictrl.dll (~779KB) UI Controls (sound control, self-view)
renderhost renderhost.exe MFC render client with Command Window, chat, object explorer
webclient webclient.exe WebBrowser host for the original 1999 HTML client
serverV2 serverV2.exe (~411KB) Server manager (MFC MDI app)
sengine sengine.exe (~400KB) Script engine (MFC MDI app)

Register & Run

All COM DLLs require elevated 32-bit regsvr32:
# Register COM DLLs (elevated command prompt)
for %d in (vwsystem vwmm vwfound vwprops vwstudio vwrendvw objexplr uictrl) do (
    C:\Windows\SysWOW64\regsvr32.exe /s build\Debug\%d.dll
)

# Set registry paths
scripts\setup_registry.bat

# Launch a world
scripts\launch_world.bat Gallery

Automation scripts:
scripts\launch_world.bat [name] — launch any world (Gallery, Home, Landscape, Office, DesktopWorld)
scripts\launch_multiuser.bat [name] — two-client multi-user test
scripts\launch_editor.bat — one-click editor with studio tools
scripts\launch_render_test.bat — build/register/create/server/render
CMake Options

Option Default Description
VWORLDS_BUILD_GRAPHICS OFF Build D3DRM/DirectDraw-dependent components (vwmm, vwstudio, vwrendvw, renderhost, webclient)
VWORLDS_BUILD_APPS ON Build sengine.exe and serverV2.exe
VWORLDS_BUILD_FOUNDATN ON Build vwfound.dll
